# Julia Website Project

## Overview
This project is a simple educational website created using HTML and CSS as part of the Website and Blog Development (JMS 467) course.

The website demonstrates basic web development concepts including page structure, navigation, styling, and linking multiple pages together.

## Features
- Multi-page website
- Navigation menu
- Course information page
- Contact page with email and phone link
- Clean and responsive layout

## Pages Included
1. Home Page
2. Course Page
3. Contact Page

## Technologies Used

- CSS
